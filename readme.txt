1.启动服务：

    在当前文件夹下打开终端，输入：

        node server.js
    
    当出现 Server listening at port 8080 即开启成功

    此时可通过  localhost + 端口号 访问，或本机ip + 端口访问。

    注：也可以直接打开相应html文件以查看。只是用启动服务的方式打开时，同一局域网的设备可以直接用本机ip+端口的方式访问到项目。


2.html标签

    标签就是一个容器，常用的，如 a标签，div标签，input标签，img标签，标签都有自己的语义，以实现特定功能。
    常用的不多，不常用的完全可以现查现用。

    元素标签内有内置属性，如a标签的href属性，img标签的src属性，属性都是由具体含义的，这些是固有属性，也可自定义属性！

    基本元素很简单，跟着网站过一遍就没问题了，这个过程可以持续5天。


3.元素内可以设置style属性，这个属性就是这个标签的css样式。暂且不表。



4.今晚达成的目的，就是要认识到html标签这个东西很简单，这都不算是程序，就是个标记的东西。完事跟着网站了解这些标签的语义就ok了。
